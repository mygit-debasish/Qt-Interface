## Description

This Qt application establishes a UART communication interface with the STM32F769I-DISCO board. It retrieves the current NTP epoch timestamp from an NTP server and transmits it to the STM32F769I-DISCO board over the UART interface. The board uses the received epoch timestamp to synchronize its Real-Time Clock (RTC).

**Reference STM32 Application:** `STM32F769I-DISCO_UART_NTP_RTC_SYNC`
